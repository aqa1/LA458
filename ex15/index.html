Alejandra Quintana
April 11th, 2019

Five Interesting Tools or Functions

1. Centroid: This tool takes one or more of the features and calculates the centroid using the mean of all vertices. By doing so it lessens the effect of small islands for example, and artifacts when calculating the centroid of a set of polygons.
2. Union: This takes two or more polygons and returns a combined polygon. But if the input polygons are not contiguous, this function returns a MultiPolygon feature. This could be beneficial when looking at two areas on a map but want to classify them as one area.
3. Line Intersect: Takes any LineString or Polygon GeoJSON and returns the intersecting point(s). This is nice to use if trying to define a midpoint on an X. Whether trying to show a specific area or locating something from two lines.
4. Mask: This takes a polygona and puts a mask on it by returning a polygon exterior ring with holes. This could be interesting when analyzing parts of a country, city or town and symbolize which part was studied.
5. HexGrid: Creates flat-topped hexagons or triangles (polygon features) that are aligned in an odd-q vertical grid that look like hexagons creating hexagon grid. This could be a cool way to present boring data.
